# fasadeteknikk.no

Norges ledende plattform for fasade, fasadearbeid og fasaderehabilitering.
En tjeneste fra [IT-Firma.no](https://it-firma.no).

## Design: Hvit / Grå / Oransje

- **Bakgrunn:** Hvit #FAFAFA / Ren hvit #FFFFFF
- **Tekst:** Koksgrå #1A1A1A
- **CTA & aksent:** Oransje #FF6B00
- **Font:** Outfit (alle vekter)
- **Stil:** Minimal, bold typografi, oransje hover-effekt på tjenesterader

## Tech stack

- **Next.js 14** – App Router, Server Components
- **TypeScript** – strikt typing
- **Tailwind CSS** – hvit/grå/oransje design system
- **Google Fonts** – Outfit
- **Vercel** – hosting

## Kjøre lokalt

```bash
npm install
npm run dev
```

## Bygge og deploye

```bash
npm run build
npx vercel --prod
```
